MicroSpringBoot IoC Framework

A minimal Java web framework demonstrating reflection-based IoC.

Dynamic Routes @RestController

  • / — Home (FirstWebService)
  • /hello — Hello (FirstWebService)
  • /greeting?name=TDSE — Greeting with @RequestParam
  • /pi — Math PI value
  • /square?n=7 — Square of a number

Static Files PNG + HTML

This page is a static HTML file served from webroot/.

How it works

  1. Framework scans classpath for @RestController classes
  2. Instantiates them using Class.newInstance()
  3. Registers @GetMapping methods as HTTP routes
  4. Resolves @RequestParam at request time via reflection
